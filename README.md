# NewsHeadlines
This repository contains a sample app that implements MVVM architecture using Dagger2, Room, RxJava, Glide, Retrofit.

# Languages, libraries and tools used
- [Kotlin](https://kotlinlang.org/docs/reference/android-overview.html)
- [MVVM](https://github.com/MindorksOpenSource/android-mvvm-architecture)
- [Retrofit2](https://www.journaldev.com/13639/retrofit-android-example-tutorial)
- [Room](https://developer.android.com/topic/libraries/architecture/room.html)
- [DataBinding](https://developer.android.com/topic/libraries/data-binding)
- [rxJava2]( https://github.com/amitshekhariitbhu/RxJava2-Android-Samples)
- [Dagger2](https://github.com/MindorksOpenSource/android-dagger2-example)
